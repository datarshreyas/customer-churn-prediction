# Customer Churn Prediction

## Objective
Predict customer churn and identify key factors influencing customer retention.

## Tools Used
Python (pandas, scikit-learn, matplotlib)

## Dataset
IBM Telco Customer Churn dataset (~7000 customers)

## Key Insights
- Customers with month-to-month contracts are more likely to churn
- Higher monthly charges are associated with increased churn risk
- Customers with longer tenure are less likely to leave
- Certain services (e.g., fiber internet) show higher churn rates

## Model Outcome
- Built a Random Forest classification model
- Evaluated performance using classification metrics

## Business Recommendations
- Encourage customers to move to long-term contracts
- Offer retention incentives for high-risk customers
- Focus on improving service quality for high-churn segments
